# brAIlle Technologies

Welcome to brAIlle Technologies - an AI-powered braille device company website.

## About

brAIlle Technologies is dedicated to bridging the accessibility gap through innovative AI-powered solutions. Our flagship product converts speech and PDFs to braille in real-time, empowering visually impaired individuals with greater independence and access to information.

## Website Features

- **Responsive Design**: Mobile-first approach with full tablet and desktop support
- **Accessibility**: WCAG compliant with ARIA labels, keyboard navigation, and semantic HTML
- **Modern UI**: Clean, professional design with gradient backgrounds and smooth animations
- **Interactive Elements**: Mobile navigation menu, smooth scrolling, and form validation

## Getting Started

### Viewing the Website Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/adiityaguru/brAIlle.git
   cd brAIlle
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser
